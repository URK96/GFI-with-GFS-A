<1.9.15 수정사항>

- Activity 전환 애니메이션 추가
- 요정 일러스트 뷰어에서 로딩바가 없어지지 않는 문제 수정
- 인형 리스트 정렬 기능 추가
- 인형 일러스트레이터, 성우 정보 추가 표시
- 인형 상세페이지에서 타이틀 바 쪽에 [총기 타입 -> 도감 번호] 로 표시되는 내용 수정
- 인형 일러스트 뷰어에 캐시 갱신 버튼 추가
- '설정'에 모든 DB 다운로드 항목 추가
- '설정'에 DB 상세페이지 배경화면 활성화 여부 항목 추가
- 소전사전v1 항목에서 업데이트 메시지가 계속 뜨는 현상 수정
- 소전사전v1 항목에서 항목 목록 리스트가 번역된 걸로 표시되도록 수정
- 모든 DB 리스트에서 리스트가 스크롤 될 때 플로팅 버튼이 사라지도록 개선
- 모든 DB 리스트에서 초기 진입시 검색란에 Focus가 맞춰지지 않도록 개선
- 모든 DB 상세페이지에서 TableLayout의 간격 조정
- 공지사항 로드에 문제가 있을 경우 에러 메시지 대신 "# 공지사항 로드 실패 #"가 표시되도록 수정
- '정보' 항목으로 진입되는 테스트 Activity 제거
